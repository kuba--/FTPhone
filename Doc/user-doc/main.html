ABOUT
FTPhone is a simple console application which try to emulate phone talk. If YOU have access to ftp server and read/write permission for any remote directory on this server, and this same abilities has your friend then you can try to talk using this program.

COMPILING AND RUNNING
You ask me how to compile and run this program ? Hmm, it's very simple. First thing, which You have to do after decompressing tar ball package (tar zxvf FTPhone.tar.gz) run a script make.sh 
 ./make.sh
After successful compiling You should be able notice executive file a.out , but do not run it if you are not sure which parameters You compiled the program. Default some sources are compiling with defined DO_DEBUG macro. Safety You should run program using script ftphone.sh, which can generate log files in directory ./Log 
 sh ftphone.sh
I suppose, that You do not know correct executive options. So i suppose that first time run it with --help option. 
 sh ftphone.sh --help 
 sh ftphone.sh [-m my user name] [-f friend user name] [-s ftp server address] [-d remote dir]

 default values:
  my user name on ftp server: kuba
  friend user name on ftp server: john
  ftp server address: ftp.server.net
  remote dir: /tmp/
eg.: sh ftphone.sh -m kuba -f john -s ftp.server.net -d /tmp/

Afterwards, you login on ftp server and if everything will be OK you'll see console interface in ncurses.





HOTKEYS

We have got only 3 hotkeys !

  • R: we press 'R' or 'r' when we want to start record yourself (program create gsm file)



  • S: we press 'S' or 's' when we want to stop recording (program sending before created gsm file)
  • Q: we press 'Q' or 'q' when we want to quit (logout and clearing temporary files)
